Trenza

Description

Trenzado de listas

Definir la función

trenza :: [a] -> [a] -> [a]

tal que (trenza xs ys) es la lista obtenida intercalando los elementos de xs e ys. Por ejemplo,

>>> trenza [5,1] [2,7,4]
[5,2,1,7]
>>> trenza [5,1,7] [2..]
[5,2,1,3,7,4]
>>> trenza [2..] [5,1,7]
[2,5,3,1,4,7]
>>> take 8 (trenza [2,4..] [1,5..])
[2,1,4,5,6,9,8,13]
